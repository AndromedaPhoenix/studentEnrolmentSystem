This project is a Student Enrolment System developed for the ITS206 Software Construction and Design course.
It demonstrates core Object‑Oriented Programming (OOP) concepts including:
- Classes & Objects
- Encapsulation
- Constructors
- Access Modifiers
- Relationships between classes
- Clean, modular design
Since my student ID ends with an odd number, I implemented a Waitlist System as the personalised extension.
